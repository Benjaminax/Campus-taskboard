Campus Task Collaboration Board is a team collaboration platform for students working on group projects. The frontend lets users register, log in, join teams, manage tasks, and view dashboards.

## Getting Started

Access the live app: [https://benjaminax.github.io/Campus-taskboard/](https://benjaminax.github.io/Campus-taskboard/)

To test quickly, use:
- Email: testuser@example.com
- Password: password123

Or register with any email and a password (min 6 characters).

## Features

- User registration/login (JWT authentication)
- Create/join teams, manage members
- Create/assign/update tasks
- Dashboard with team/task statistics

## Local Development

1. Clone this repository.
2. Open `index.html` in your browser.

## Technologies

- HTML, CSS, JavaScript (vanilla)
- Communicates with backend API (see backend repo)

## Deployment

Hosted on GitHub Pages. See the backend repo for API details.
